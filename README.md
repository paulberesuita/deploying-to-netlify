# deploying-to-netlify



## Local Project Setup

``` bash
# install vue cli
$ npm install vue-cli -g

# create nuxt starter
$ vue init nuxt/starter deployapp

# navigate to app
$ cd deployapp

# install deployapp dependencies
$ npm install

```

## Setup app for Github

``` bash
# initializes git
git init

# stages all files to be committed 
git add .

# commit (locally)
git commit -m "Initial commit"

```
